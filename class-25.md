## nosql vs sql

### SQL

- predefined schema
- relational database
- vertically scalable
- table based databse
- not the best fit for hierarchical data storage

### noSQL

- dynamic schema for unstructured data
- non-relational database
- horizonally scalable
- document based
- good fit for hierarchical data storage
